Recent Work

SchoolScale 2.0

SchoolScale is a Responsive CRUD Server-Side Web Application built in MVC C#/ASP.NET Core, that gives a user the perfect place research higher education options to avoid overpaying for higher education. Used as a supplement to other qualitative search methods users can weigh school choices quickly comparing projected career earnings with multiple loan scenarios to make more informed decisions. SchoolScale utilizes a SQL Server Database that is accessed through Entity Framework, and makes use of BootStrap 4 and JavaScript for styles and DOM manipulation.


Technologies Used: C#/ASP.Net Core, MVC, SQL Server, Entity Framework, Identity Framework, Bootstrap, HTML, CSS, JavaScript, Moqups for Wireframes, GitHub Projects with version control via Git and GitHub Application design, dbdiagram.io for ERD to define data relationships.

SchoolScale 1.0

SchoolScale is a single page responsive CRUD web application that solves important problems for people who are looking at higher education schools and colleges.  SchoolScale provides users a concise and informative cost benefit analysis of their school options. The app takes users thru a menu of degree / program options they can choose from and presents the users with career earnings research and tuition costs and give them the opportunity to enter savings, grants and scholarship data into the app so they can generate a Benefit/ Cost Ratio Analysis which reveals the long term benefits of each potential Education Plan in terms of funds spent and earnings to be received over time in the chosen profession.


Technologies used: React, HTML, CSS, JavaScript, Moqups for Wireframes, JSON, GitHub Projects with version control via Git and GitHub Application design, dbdiagram.io for database planning the ERD to define data relationships.
